Installation sous Windows

Sous Windows, le logiciel client ne nécessite aucune installation. Les bibliothèques nécessaires à l'exécution du logiciel sont présentes dans le dossier "Windows-Client", il suffit de lancer SPM_Client.exe pour ouvrir le logiciel client.

Par défaut, les fichiers écrits par le logiciel client sont placés dans %USERPROFILE%\spm\

Installation sous Unix

Sous Unix, le logiciel client nécessite des bibliothèques spécifiques présentes dans le dossier "usr lib - Client" à copier dans /usr/lib :
  • libicu*
  • libQt5*

D'autres bibliothèques spécifiques sont présentes dans le dossier "Linux-Client" et sont à conserver à côté de l'exécutable "SPM_Client".

Par défaut, les fichiers écrits par le logiciel client sont placés dans HOME/spm/
