Informations

Le logiciel Secure Password Manager a été développé dans le cadre des projets tuteurés de l'Institut Universitaire de Technologie Informatique de l'université Paul Sabatier à Toulouse.

Ses auteurs sont :
  • Yoann Fleytoux
  • Sylvain Laugié
  • Alexandre Schreiber
  • Olivier Sire de Vilar

Le projet a été développé à destination de Boris Valera, sous le tutorat de Nadia Yassine-Diab.

Le logiciel Secure Password Manager a été développé en utilisant l'environnement de développement intégré QtCreator 2.8.1 et fait usage des bibliothèques Qt 5.1.1 et OpenSLL 1.0.1e.
  • Qt Project
  • OpenSLL Project
  • OpenSSL pour Windows
